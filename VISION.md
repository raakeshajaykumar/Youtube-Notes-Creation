Goal:
Convert educational content into textbook quality learning material.

Principles:
- Transcript is not authoritative.
- AI must verify claims.
- Official sources have highest priority.
- Expand beyond source material.
- No page limit.
- Include screenshots and diagrams.
- Domain independent.
- Prefer correctness over speed.

Pipeline

1. Source Acquisition
   - YouTube videos
   - Playlists
   - PDFs
   - Websites
   - Documentation
   - Books

2. Transcript Extraction
   - Extract transcript
   - Timestamp preservation
   - Speaker detection where possible

3. Transcript Cleaning
   - Remove filler
   - Remove sponsor segments
   - Remove repetition

4. Claim Extraction
   - Extract factual claims
   - Extract concepts
   - Extract definitions

5. Verification Engine
   - Verify claims against authoritative sources
   - Flag contradictions
   - Assign confidence score

6. Knowledge Expansion
   - Add explanations
   - Add examples
   - Add prerequisites
   - Add advanced context

7. Visual Enrichment
   - Official screenshots
   - Diagrams
   - Architecture figures
   - Tables

8. Learning Material Generation
   - Exercises
   - Labs
   - Interview questions
   - Flashcards

9. Book Assembly
   - Chapters
   - References
   - Cross links
   - Index

10. Export
   - PDF
   - EPUB
   - Website
   - Obsidian vault
